# MAIDGAR
Multimodal AI-driven Debate Graph and Attack Reasoner (MAIDGAR) is LLM+KG representing real and debated facts, in several forms from text to video, using Fuzzy Knowledge Graphs, with 5-10% of facts are debated by vetted political and/or community. It helps disambiguate and detect disinformation and fraud, anticipate attacks, and advise mitigation.
